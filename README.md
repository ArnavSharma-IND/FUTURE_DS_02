# 📉 Customer Retention & Churn Analysis

An interactive analytics dashboard for understanding customer churn, retention patterns, and subscription behavior using the Telco Customer Churn dataset.

## 🚀 Features

* Customer Churn Analysis
* Retention Analysis
* Customer Lifetime Insights
* Contract Analysis
* Payment Method Analysis
* Interactive Filtering

## 📁 Dataset Overview

The dataset contains customer demographics, subscription details, and churn information.

### Key Columns

* customerID
* tenure
* Contract
* InternetService
* PaymentMethod
* MonthlyCharges
* TotalCharges
* Churn

## 📈 KPIs

* Total Customers
* Active Customers
* Churned Customers
* Churn Rate
* Average Customer Lifetime

## 📊 Visualizations

* Churn Distribution
* Contract-wise Churn
* Internet Service Analysis
* Customer Lifetime Distribution
* Payment Method Analysis

## 🛠️ Technologies Used

* Python
* Pandas
* Plotly
* Streamlit

## ▶️ Run the Dashboard

```bash
streamlit run app.py
```

## 💡 Business Insights

* Long-term contracts reduce churn.
* Certain payment methods show higher churn.
* Customer tenure strongly influences retention.
* Internet services impact customer satisfaction.
